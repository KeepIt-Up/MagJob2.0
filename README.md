## Development
- Install Docker Desktop
- Update Google IDP `"clientSecret": "**********"` in `src/Keycloak/realm-export.json`
- Run the following command to start the application:

```bash
git clone https://github.com/KeepIt-Up/MagJob2.0.git
cd MagJob2.0
docker compose up
```
